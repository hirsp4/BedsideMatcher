996  V1 Call:
997  Sets the broadcast data.
998  dynData contains flags that enable the replacement of sections in the given data with dynamically created data
999  like battery level or temperature.
1000  dynOfs is the offset at which the insertion of the dynamic data takes place.
1001 
1002  The device can transmit up to 4 different broadcast messages, identified by the index (0 to 3).
1003  Calling with nil as data will disable the broadcast message.
1004 
1005  The user is responsible for implementing a way of differentiating among the different broadcast messages on reception.
1006 
1007  The maximum data length is 29 Bytes.
1008 
1009  */
