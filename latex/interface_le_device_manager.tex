\subsection{Detailed Description}
\hyperlink{interface_le_device_manager}{Le\+Device\+Manager} is responsible for handling the management of the C\+B\+Central\+Manager class The main purposes include\+:
\begin{DoxyItemize}
\item Scanning for B\+L\+E peripherals
\item Associating compatible peripherals with the device classes included in this S\+D\+K (e.\+g. \hyperlink{interface_le_snf_device}{Le\+Snf\+Device})
\item Creating and storing a list of visible S\+D\+K devices 
\end{DoxyItemize}
